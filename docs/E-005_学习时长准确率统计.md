# E-005 学习时长、准确率数据统计功能

## 功能概述

本功能实现了用户学习数据的完整统计系统，包括：

- **练习时长追踪**: 自动记录每次练习的持续时间
- **准确率统计**: 基于实时姿态检测计算并记录准确率
- **总学习时长**: 累计所有练习时间
- **平均准确率**: 计算所有练习的平均准确率
- **掌握技法数量**: 统计准确率达到 80% 以上的技法数量

## 数据库模型

### PracticeRecord (练习记录表)
```sql
- id: 主键
- user_id: 用户 ID
- craft_id: 工艺 ID
- craft_name: 工艺名称
- scenario: 场景 (embroidery/clay)
- duration: 练习时长 (秒)
- score: 得分 (0-100)
- accuracy: 准确率 (0-100)
- feedback: 反馈信息
- completed_at: 完成时间
```

### User (用户表)
```sql
- id: 主键
- name: 用户姓名
- avatar_url: 头像 URL
- level: 等级 (beginner/apprentice/advanced/master/grandmaster)
- experience_points: 经验值
- title: 称号
```

### UserAbility (用户能力表)
```sql
- user_id: 用户 ID (唯一)
- stability: 稳定性 (0-100)
- accuracy: 准确度 (0-100)
- speed: 速度 (0-100)
- creativity: 创造力 (0-100)
- knowledge: 知识 (0-100)
```

## API 端点

### 获取用户档案
```
GET /api/v1/user/profile?user_id={userId}
```

返回:
```json

  "user": {
    "id": 1,
    "name": "李明",
    "level": "apprentice",

  "abilities": {

  },
  "stats": {
    "total_practice_hours": 12.5,
    "average_accuracy": 85,

```

### 获取统计数据
```
GET /api/v1/user/profile/{userId}/stats
```

### 添加练习记录
```
POST /api/v1/user/profile/{userId}/practice-record
```

参数:
- `craft_id`: 工艺 ID
- `craft_name`: 工艺名称
- `duration`: 时长 (秒)
- `score`: 得分
- `accuracy`: 准确率
- `scenario`: 场景 (可选)
- `feedback`: 反馈 (可选)

## 使用方法

### 前端使用练习追踪器

```javascript
import { tracker, savePracticeRecord } from '../utils/practiceTracker';

// 开始练习
tracker.start('embroidery', 'suzhou_embroidery', '苏绣·平针基础');

// 记录准确率采样
tracker.recordAccuracy(85);
tracker.recordAccuracy(90);

// 结束练习
const stats = tracker.end();
// stats = {
//   duration: 300,  // 秒
//   accuracy: 87.5, // 平均准确率
//   score: 87.5,
//   scenario: 'embroidery',
//   craftId: 'suzhou_embroidery',
//   craftName: '苏绣·平针基础'
// }

// 保存到后端
await savePracticeRecord(userId, stats);
```

### 获取统计数据

```javascript
import { getUserStats, getUserProfile } from '../utils/practiceTracker';

// 获取统计数据
const stats = await getUserStats(userId);
// {
//   total_practice_hours: 12.5,
//   average_accuracy: 85,
//   mastered_crafts: 3,
//   total_works: 15
// }

// 获取完整档案
const profile = await getUserProfile(userId);
```

准确率基于手部关键点的距离是否在目标范围内：

```javascript
// 苏绣阈值
const thresholds = { min: 0.03, max: 0.15 };

```

// 基于分数和时长计算经验值
